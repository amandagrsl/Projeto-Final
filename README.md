# Projeto-Final

## Sistema de Imobiliária

Este projeto foi desenvolvido como trabalho final do curso de Tecnologia em Análise e Desenvolvimento de Sistemas, com o objetivo de aplicar na prática os conhecimentos adquiridos ao longo da formação.

Trata-se de um sistema web para gerenciamento de uma imobiliária, permitindo o controle de usuários e imóveis de forma dinâmica e funcional.

---

## Funcionalidades

- Cadastro de usuários  
- Sistema de autenticação (login e logout)  
- Cadastro e gerenciamento de imóveis  
- Integração com banco de dados  
- Visualização de imóveis disponíveis  
- Integração com mapa para exibição da localização real dos imóveis  

---

## Diferencial

Um dos principais destaques do projeto é a integração com a Google Maps API, que permite visualizar os imóveis diretamente em um mapa com localização real.

Essa funcionalidade torna a aplicação mais interativa e aproxima o sistema de soluções utilizadas no mercado.

---

## Tecnologias Utilizadas

- PHP  
- MySQL  
- HTML  
- CSS  
- JavaScript  

---

## Objetivo do Projeto

Desenvolver uma aplicação completa que simule um sistema real do setor imobiliário, colocando em prática conceitos fundamentais de desenvolvimento web, como organização de código, manipulação de dados e integração entre front-end e back-end.

---

## Aprendizados

Durante o desenvolvimento deste projeto, foi possível aprofundar conhecimentos em:

- Estruturação de sistemas web  
- Modelagem de banco de dados  
- Desenvolvimento de operações CRUD  
- Integração entre front-end e back-end  
- Organização de lógica de negócio  

---

## Como executar o Projeto

1. Clone este repositório: https://github.com/amandagrsl/Projeto-Final
   
2. Configure o banco de dados MySQL  

3. Inicie um servidor local (XAMPP, WAMP ou similar)  

4. Acesse no navegador: http://localhost/Projeto-Final
   
---

## Observação

Atualmente, o sistema roda em ambiente local, sendo necessário configurar um servidor local para execução.
